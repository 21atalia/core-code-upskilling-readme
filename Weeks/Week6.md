_1. [Expressions Matter]() exercise_

> ### Solution

```js
SELECT
(
  SELECT MAX(mx)
  FROM 
  ( VALUES (a+b+c),
           (a+b*c),
           (a*b+c),
           (a*b*c),
           (a*(b+c)),
           ((a+b)*c) )
  AS _(mx)
) AS res
FROM expression_matter;
```
